# anfr-supports-to-geojson
URL : https://ed94120.github.io/anfr-supports-to-geojson/

L'application convertit en fichier geoJSON un fichier support d'antenne en .txt extrait du fichier ZIP de l'ANFR.

Le fichier ZIP est récupéré sur cet URL : https://www.data.gouv.fr/datasets/donnees-sur-les-installations-radioelectriques-de-plus-de-5-watts-1

Le fichier geoJSON créé doit ensuite être placé dans le dossier data du repository cartes-ign-topographie et le nom du fichier mis à jour dans le fichier index.html
