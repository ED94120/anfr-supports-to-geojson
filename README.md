# anfr-supports-to-geojson
URL : https://ed94120.github.io/anfr-supports-to-geojson/

L'application convertit en fichier geoJSON le fichier support d'antennes en .txt qui contient les localisations de toutes les antennes-relais et est extrait du fichier ZIP de l'ANFR.

Le fichier ZIP ANFR est récupéré à partir de cet URL : https://www.data.gouv.fr/datasets/donnees-sur-les-installations-radioelectriques-de-plus-de-5-watts-1

Le fichier geoJSON créé doit ensuite être placé dans le dossier data du repository cartes-ign-topographie

Le nom du fichier doit mis à jour dans le fichier index.html du repository cartes-ign-topographie
